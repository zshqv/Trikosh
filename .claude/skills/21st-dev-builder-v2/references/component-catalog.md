## How to Discover New Components

1. **Browse category**: `https://21st.dev/s/{slug}` — shows all components sorted by popularity
2. **Search specific**: Use WebSearch with `site:21st.dev {keywords}`
3. **Author pages**: `https://21st.dev/{author}` — see all of an author's components
4. **Community**: `https://21st.dev/community/components/search` — semantic search
5. **Check for new categories**: Browse `https://21st.dev/home` for the full category listing
